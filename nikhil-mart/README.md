# Nikhil Mart

A grocery delivery web application inspired by Swiggy Instamart. Built with React + Vite.

## Features

- Home page with hero banner, promotional offers, category grid, and product sections
- Products page with category filtering, search, and sorting
- Shopping cart with quantity management and checkout
- Responsive design for mobile and desktop

## Tech Stack

- React 19
- Vite
- React Router v7
- React Icons

## Getting Started

```bash
cd nikhil-mart
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

```bash
npm run build
```

Output will be in the `dist/` directory.
